# ✈️ Flight Radar

Gerçek zamanlı uçuş takip uygulaması. Dünyadaki uçuşları harita üzerinde görüntülemek, uçuş detaylarını incelemek ve rotaları takip etmek için tasarlanmış modern ve interaktif bir web uygulamasıdır.

## 🛠️ Kullanılan Teknolojiler

- **React 19** - Modern UI geliştirme
- **Vite** - Hızlı build ve development
- **Redux Toolkit** - State management
- **React Leaflet** - İnteraktif harita görüntüleme
- **Axios** - API istekleri
- **React Router DOM** - Sayfa yönlendirme
- **Bootstrap** - Responsive tasarım
- **Lucide React** - İkonlar
- **RapidAPI (Flight Radar API)** - Gerçek zamanlı uçuş verileri

## ⭐ Özellikler

- 🗺️ Gerçek zamanlı uçuşların harita üzerinde görüntülenmesi
- ✈️ Uçak konumlarının dinamik olarak güncellenmesi
- 📍 Kalkış ve varış noktalarının işaretlenmesi
- 🛤️ Uçuş rotalarının harita üzerinde çizilmesi
- 📊 Liste ve harita görünümü arasında geçiş
- 🔍 Detaylı uçuş bilgileri (callsign, rota, havaalanı bilgileri)
- 📱 Responsive ve mobil uyumlu tasarım
- 🎯 Uçağa tıklayarak detaylı bilgi görüntüleme
- 🌍 OpenStreetMap entegrasyonu
- 🔄 Dinamik uçak yönlendirme ikonları
